# PRODIGY_WD_02

## Stopwatch Web Application

This project was developed as part of Task 02 of the Web Development Internship at Prodigy InfoTech.

### Description

A stopwatch web application built using HTML, CSS, and JavaScript that allows users to start, pause, reset, and record lap times accurately.

### Features

* Start the stopwatch
* Pause the stopwatch
* Reset the stopwatch
* Record lap times
* Responsive and user-friendly interface

### Technologies Used

* HTML
* CSS
* JavaScript

### Learning Outcomes

* Understanding JavaScript timing functions
* DOM manipulation
* Event handling
* Dynamic content updates
* Building interactive web applications
